# Ignis3D-Portifolio
Ignis3D é a aplicação principal do ecossistema Ignis, desenvolvida para modelagem 3D interativa e criativa.
Ela permite importar modelos em diversos formatos no mesmo visualizador ou separado (OBJ, GLTF, GLB, NIF);
voxelizá-los e exportar para formatos compatíveis (JSON) com Minecraft Java (ItemsAdder) e/ou Minecraft Bedrock com suporte a texturas, materiais metálicos e vidro e configuração flexível via arquivo YAML (voxelizer.yml);
Exportar para outros formatos como OBJ, GLB, GLTF, NIF ou o formato próprio I3D totalmente configurável.
Interface estilizada com temas customizáveis via QSS e com painel para configuração flexível.

Desenvolvido por João Gabriel Maier (JGMaier) como parte do portfólio Ignis3D.

Frameworks utilizados: PyQt5/PySide2, Trimesh, NumPy, PIL.
